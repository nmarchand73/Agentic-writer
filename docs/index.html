CopilotKit · AG-UI · Pydantic AI

Agentic Writer

Automated fiction pipeline (Writer → Editor → markdown → docx/PDF). The Studio uses CopilotKit v2 and AG-UI for generative UI—live pipeline state, not chat-only. Same flow via CLI for headless runs.

Stack

  • Python — Pydantic AI, FastAPI /agui, Typer CLI
  • Next.js — CopilotKit runtime, persisted threads
  • Skills — story-writer, manuscript-editor, print-layout
  • Tests — Gherkin BDD (pytest-bdd), CI without OpenAI

Quick start

  • uv sync --all-extras · npm install
  • cp .env.example .env · set OPENAI_API_KEY
  • CLI: uv run agentic-writer generate …
  • Studio: uv run agentic-writer serve + cd web && npm run dev
