From pitch to print-ready fiction—one pipeline, two ways to run it

Agentic Writer orchestrates a Writer → Editor flow, saves structured artefacts, and exports docx & PDF. Use the CLI in CI or batch jobs; use the Studio when you want a live UI that shows pipeline progress—not just chat bubbles.

Why this stack (2025–2026)

Deliberate choices for agent UX, a clean backend boundary, and tests that run without burning API credits.

  • CopilotKit v2 Studio UI

    Multi-route runtime (/api/copilotkit), thread list, and chat—so you build story tools and state, not another SSE/WebSocket protocol from scratch.

  • AG-UI Agent wire format

    Standard events from FastAPI POST /agui to the browser: STATE_SNAPSHOT and STATE_DELTA drive generative UI (steps, slug, errors) while tools run—not a wall of text.

  • Pydantic AI Python agents

    Typed Writer and Editor agents, skill-backed prompts, and a Studio orchestrator with tools (run_story_generation). Same run_pipeline() as the CLI.

  • FastAPI + Next.js Split deploy

    Python owns the model and filesystem; Node owns React and CopilotKit. Manuscript and PDF are served via REST so large markdown never bloats agent state.

  • Gherkin + pytest-bdd Quality

    Executable specs per slice (env, CLI, pipeline, Studio API, threads). CI markers run with mocked agents—no OpenAI key required on every PR.

Generative UI, not chat-only

The Studio reflects what the pipeline is doing—like a build dashboard for fiction.

Shared agent state

StudioState carries slug, step checklist, and errors. The React tree updates on AG-UI deltas while run_story_generation is still running.

Progress & deliverables

StudioProgressBridge streams step updates. Tabs load final markdown and PDF from /manuscript/{slug} and /pdf/{slug}. Threads persist under .data/studio-threads/.

CLI or Studio

One pipeline implementation; pick the surface that fits your workflow.

CLI (Typer + uv)

Headless generate, serve, doctor. Ideal for scripts, batch runs, and automation.

  1. uv sync --all-extras
  2. uv run agentic-writer generate …

Studio (local)

CopilotKit chat + generative pipeline UI. Requires API key and both servers on your machine.

  1. ./scripts/run.sh
  2. or npm run studio
